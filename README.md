# CloudMusic-mobile
## 基于Vue3+Vite+TypeScript+Node.js+ElementPlus的网易云音乐移动端 (功能待完善)

## 项目初始化
npm init

## 项目启动
npm run dev

## 项目编译
npm run bulid


## 基本功能
* banner图获取
* 推荐歌单
* 歌单详细页
* 播放功能
* 歌曲切换/暂停
* 歌曲搜索
* 历史记录
...其他功能待开发
